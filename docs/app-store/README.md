# Publicação iOS - ICA Telemedicina Humana

## Arquivos desta pasta
- `app-store-connect-metadata.md`: textos prontos para App Store Connect.
- `assets-checklist-ios.md`: checklist de screenshots e ícones.
- `testflight-submission-runbook.md`: passo a passo de build e envio.
- `review-rejection-playbook.md`: respostas e ações para rejeições comuns.

## Comandos principais do projeto
```bash
npm run prepare:web
npm run cap:sync:ios
npm run cap:open:ios
```
